# Divider

The `@nx-kit/divider` package contains an accessible Divider component.

## Installation
---

```text
npm install @nx-kit/divider
```

## Usage
---

<Divider skin="100" />

```javascript
import { Divider } from '@nx-kit/divider';

<Divider skin="100" />
```

More examples in [Storybook](https://nexumag.github.io/nx-kit/?path=/story/nx-kit-divider--horizontal).

## Theming
---

The Divider can be styled by its styled property `orientation`.

Have a look at the default theme: https://github.com/nexumAG/nx-kit/blob/master/packages/libs/%40nx-kit/theme-default/src/components/divider.ts
