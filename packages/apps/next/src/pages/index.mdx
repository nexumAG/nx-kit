# @nx-kit

React component library and design system

**Storybook:** [https://nexumag.github.io/nx-kit](https://nexumag.github.io/nx-kit)

## Introduction
---

nx-kit is a collection of components and helpers for React.
They all live in their own packages and are published separately to npm.

The components are optimized for accessibility with the help of https://react-spectrum.adobe.com/react-aria/.

https://styled-components.com/ is used for the theming and styling.
There is also a built-in design system similar to https://styled-system.com/.
