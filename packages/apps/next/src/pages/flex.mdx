# Flex

The `@nx-kit/flex` package contains a Flex component which you can use as a layout container or grid.

It is similar to the View component but `display` is set to `flex` by default and can be changed with the `flexType` prop.

The flex properties like `justifyContent` and `flexFlow` are available as direct props, the other styles under `styles`.

In addition it has the special props `col`, `row` and `colOffset` which is the row/column size or offset in float percentage value.

The `gap` prop is a polyfill of the native `gap` property. In newer browsers you can also use `gapNative`.

## Installation
---

```text
npm install @nx-kit/flex
```

## Usage
---

<Flex justifyContent="space-between" styles={{ display: { xs: 'block', md: 'flex' }, backgroundColor: '#f0f0f0', padding: '20px' }}>
  <div>Col1</div>
  <div>Col2</div>
  <div>Col3</div>
</Flex>

```javascript
import { Flex } from '@nx-kit/flex';

<Flex
  justifyContent="space-between"
  styles={{ display: { xs: 'block', md: 'flex' }, backgroundColor: '#f0f0f0', padding: '20px' }}
>
  <div>Col1</div>
  <div>Col2</div>
  <div>Col3</div>
</Flex>
```

Usage of `col` and `gap`:

<Flex
  gap={{ xs: '5px', sm: '15px', md: '15px 30px' }}
  flexWrap="wrap"
  flexDirection={{ xs: 'column', sm: 'row' }}
>
  <Flex col={{ sm: 1 / 2, md: 1 / 3 }} styles={{ backgroundColor: '#f0f0f0', padding: '20px' }}>Col</Flex>
  <Flex col={{ sm: 1 / 2, md: 1 / 3 }} styles={{ backgroundColor: '#f0f0f0', padding: '20px' }}>Col</Flex>
  <Flex col={{ sm: 1 / 2, md: 1 / 3 }} styles={{ backgroundColor: '#f0f0f0', padding: '20px' }}>Col</Flex>
</Flex>

```javascript
import { Flex } from '@nx-kit/flex';

<Flex
  gap={{ xs: '5px', sm: '15px', md: '15px 30px' }}
  flexWrap="wrap"
  flexDirection={{ xs: 'column', sm: 'row' }}
>
  <Flex col={{ sm: 1 / 2, md: 1 / 3 }} styles={{ backgroundColor: '#f0f0f0', padding: '20px' }}>
    Col
  </Flex>
  <Flex col={{ sm: 1 / 2, md: 1 / 3 }} styles={{ backgroundColor: '#f0f0f0', padding: '20px' }}>
    Col
  </Flex>
  <Flex col={{ sm: 1 / 2, md: 1 / 3 }} styles={{ backgroundColor: '#f0f0f0', padding: '20px' }}>
    Col
  </Flex>
</Flex>
```

More examples in [Storybook](https://nexumag.github.io/nx-kit/?path=/docs/nx-kit-flex--default).

## Theming
---

The Flex component does not have a skin or global CSS available in the theme. You can style it with the `styles` props or by extending it with `styled`:

```javascript
import { Flex } from '@nx-kit/flex';
import { styled } from '@nx-kit/styling';

const FlexStyled = styled(Flex)`
  // your special CSS
`;
```
